# Keyboard Shortcuts

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Open file |
| `Ctrl+E` | Export filtered data |
| `Ctrl+S` | Save session |
| `Ctrl+Shift+O` | Load session |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+W` | Close active tab |
| `Ctrl+1-9` | Switch to tab 1-9 |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |

## Search & Filter

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Focus search bar |
| `Ctrl+Shift+F` | Cross-tab search |
| `Ctrl+B` | Toggle bookmarked rows only |
| `Escape` | Clear search / close modal |

## Grid

| Shortcut | Action |
|----------|--------|
| `Ctrl+R` | Reset column layout |
| `Click` | Select row |
| `Shift+Click` | Range select rows |
| `Ctrl+Click` | Context menu |

## Tools

| Shortcut | Action |
|----------|--------|
| `Ctrl+H` | Toggle histogram |

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl+Q` | Quit |
| `F11` | Toggle fullscreen |

## Context Menu Shortcuts

`Ctrl+Click` or **right-click** any cell or column header to access additional actions:

- Copy cell / Copy row
- Bookmark / Tag
- Pin column
- Group by column
- Filter / Exclude value
- Auto-fit column width
- Create color rule
